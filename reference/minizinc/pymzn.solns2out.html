pymzn.solns2out¶

pymzn.solns2out(solns_input, ozn_file)¶

Wraps the solns2out utility, executes it on the input solution stream, and then returns the output.

Parameters:
  • solns_input (str) – The solution stream as output by the solver, or the content of a solution file
  • ozn_file (str) – The ozn file path produced by the mzn2fzn utility
Returns:

A list of solutions as strings. The user needs to take care of the parsing. If the output is in dzn format one can use the parse_dzn function.
Return type:

list of str
